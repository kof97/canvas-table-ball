# canvas-table-ball

canvas 实现的双人桌球小游戏

地址：

> http://kof666.com/table-ball/

---

直接运行 `index.html`

## 操作

`点击图标` 开始或按 `空格键` 开始

玩家A：

`w`、`s` 操纵上下

玩家B：

`上`、`下` 按键操纵上下

没接到球则对面得分，空格或点击图标继续

---

## 文件说明

`src` 为开发目录，`dist` 为资源目录

开发前先运行

```
	npm install
```

安装所需依赖

然后运行

```
	gulp watch
```

监听文件变化并编译
